# Typography

The design system uses Inter from Google Fonts with a clear typographic hierarchy. All text styles follow design tokens for consistent sizing, weight, and spacing.

<ComponentPreview>
  <TypographyPreview />
</ComponentPreview>

## Usage

```tsx
import { H1, H2, H3, H4, P, Strong, Small, Muted, Code, Link } from '@e-infra/design-system'

<H1>Main Heading</H1>
<H2>Section Heading</H2>
<P>Standard body text</P>
```

## Typography Scale

| Element | Tag | Font Size | Line Height | Font Weight |
|---------|-----|-----------|-------------|-------------|
| H1 | `h1` | `text-4xl` (36px) | `leading-none` (1.1) | `font-bold` |
| H2 | `h2` | `text-3xl` (30px) | `leading-none` (1.2) | `font-semibold` |
| H3 | `h3` | `text-2xl` (24px) | `leading-none` (1.3) | `font-semibold` |
| H4 | `h4` | `text-xl` (20px) | `leading-tight` (1.4) | `font-semibold` |
| P | `p` | `text-base` (16px) | `leading-7` (22px) | `font-normal` |
| Lead | `p` | `text-lg` (18px) | `leading-8` (24px) | `font-normal` |
| Strong | `strong` | `text-base` (16px) | `leading-7` (22px) | `font-semibold` |
| Small | `small` | `text-sm` (14px) | `leading-6` (18px) | `font-normal` |
| Muted | `muted` | `text-xs` (12px) | `leading-5` (15px) | `font-normal` |
| Code | `code` | `text-sm` (14px) | `leading-normal` | `font-mono` |

## Text Styles

### Headings

```tsx
<H1>Main Heading - text-4xl font-bold tracking-tight</H1>
<H2>Section Heading - text-3xl font-semibold tracking-tight</H2>
<H3>Sub-heading - text-2xl font-semibold tracking-tight</H3>
<H4>Sub-section - text-xl font-semibold</H4>
```

### Body Text

```tsx
<P>Standard paragraph text for body content.</P>
<Lead>Larger intro paragraph for emphasizing key messages.</Lead>
<Strong>Bold paragraph for important information.</Strong>
```

### Meta Text

```tsx
<Small>Caption text for secondary information.</Small>
<Muted>Extra small muted text for timestamps and labels.</Muted>
<Code>const greeting = "Hello World";</Code>
```

### Links

```tsx
<Link href="#">Standard link with underline</Link>
```
