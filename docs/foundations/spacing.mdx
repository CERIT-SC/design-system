# Spacing

The design system uses a 4px-based spacing scale for consistent layout and alignment. All spacing follows a predictable pattern for predictable spacing between elements.

<ComponentPreview>
  <SpacingPreview />
</ComponentPreview>

## Usage

```tsx
/* Use Tailwind spacing utilities with the 4px scale */
<div className="p-4">8px padding</div>
<div className="p-8">16px padding</div>
<div className="p-12">24px padding</div>
<div className="p-16">32px padding</div>
<div className="p-24">48px padding</div>
<div className="p-32">64px padding</div>
<div className="p-48">96px padding</div>
<div className="p-64">128px padding</div>

/* Responsive spacing */
<div className="p-4 md:p-8 lg:p-16">
```
