# Sidebar

Sidebar is a composable navigation system built from primitives. It supports responsive mobile sheet behavior, desktop collapse modes, grouped navigation, submenus, badges, skeleton loaders, and an optional inset content area.

## Usage

```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarHeader,
  SidebarInset,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem,
  SidebarProvider,
  SidebarSeparator,
  SidebarTrigger,
} from '@e-infra/design-system'

<SidebarProvider>
  <div className="flex min-h-screen w-full">
    <Sidebar>
      <SidebarHeader>
        <div className="px-4 py-2 font-semibold">App</div>
        <SidebarSeparator />
      </SidebarHeader>
      <SidebarContent>
        <SidebarGroup>
          <SidebarGroupLabel>Navigation</SidebarGroupLabel>
          <SidebarGroupContent>
            <SidebarMenu>
              <SidebarMenuItem>
                <SidebarMenuButton asChild>
                  <a href="/dashboard">
                    <span>Dashboard</span>
                  </a>
                </SidebarMenuButton>
              </SidebarMenuItem>
            </SidebarMenu>
          </SidebarGroupContent>
        </SidebarGroup>
      </SidebarContent>
    </Sidebar>
    <SidebarInset>
      <header className="flex h-16 items-center border-b px-4">
        <SidebarTrigger />
      </header>
      <main className="p-4">Main content</main>
    </SidebarInset>
  </div>
</SidebarProvider>
```

## Components

| Component | Description |
|-----------|-------------|
| `SidebarProvider` | Context provider for sidebar state, mobile state, and toggle behavior |
| `Sidebar` | Main sidebar container |
| `SidebarTrigger` | Toggle sidebar visibility |
| `SidebarRail` | Clickable rail used for resize/toggle interactions |
| `SidebarInset` | Main content area that adapts to sidebar state |
| `SidebarHeader` | Header area inside sidebar |
| `SidebarFooter` | Footer area inside sidebar |
| `SidebarSeparator` | Sidebar-specific separator |
| `SidebarInput` | Compact input style for sidebar search/filter |
| `SidebarContent` | Content area |
| `SidebarGroup` | Group container inside content |
| `SidebarGroupLabel` | Label for group sections |
| `SidebarGroupAction` | Action button for a group |
| `SidebarGroupContent` | Wrapper for group body |
| `SidebarMenu` | Menu container |
| `SidebarMenuItem` | Menu item |
| `SidebarMenuButton` | Menu button |
| `SidebarMenuAction` | Item-level action button |
| `SidebarMenuBadge` | Badge for counts/status |
| `SidebarMenuSkeleton` | Skeleton placeholder for loading states |
| `SidebarMenuSub` | Submenu |
| `SidebarMenuSubItem` | Submenu item |
| `SidebarMenuSubButton` | Submenu button |

## Props

### SidebarProvider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| defaultOpen | `boolean` | `true` | Initial open state (uncontrolled mode) |
| open | `boolean` | - | Controlled desktop open state |
| onOpenChange | `(open: boolean) => void` | - | Controlled state callback |
| className | `string` | - | Additional classes on wrapper |
| style | `React.CSSProperties` | - | Style overrides (supports sidebar width vars) |
| ...props | `React.ComponentProps<"div">` | - | Native `div` props |

### Sidebar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| side | `left` \| `right` | `left` | Sidebar side |
| variant | `sidebar` \| `floating` \| `inset` | `sidebar` | Sidebar visual variant |
| collapsible | `offcanvas` \| `icon` \| `none` | `offcanvas` | Collapse behavior |
| className | `string` | - | Additional CSS classes |
| children | `React.ReactNode` | - | Sidebar content |
| ...props | `React.ComponentProps<"div">` | - | Native `div` props |

### SidebarTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| onClick | `(event) => void` | - | Optional click handler (runs before internal toggle) |
| ...props | `React.ComponentProps<typeof Button>` | - | Button props |

## Variants

| Variant | Description |
|---------|-------------|
| `sidebar` | Standard docked sidebar |
| `floating` | Sidebar with rounded panel and shadow treatment |
| `inset` | Inset layout style coordinated with `SidebarInset` |

## Collapsible Options

| Option | Behavior |
|--------|----------|
| `none` | Always visible |
| `icon` | Collapse to icon-width rail |
| `offcanvas` | Slide out completely |

## Behavior

| Behavior | Details |
|----------|---------|
| Mobile mode | Renders as a `Sheet` using `openMobile` state from context |
| Desktop mode | Uses `data-state`, `data-collapsible`, `data-variant`, and `data-side` for styling |
| Keyboard shortcut | `Cmd/Ctrl + b` toggles sidebar |
| Persisted state | Open/closed state is stored in cookie `sidebar_state` for 7 days |
| Tooltip support | `SidebarMenuButton` can show tooltips when collapsed |
