## Usage

```tsx

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| logo | string \| {src: string, width?: number, height?: number} | e-INFRA default | Logo source |
| logoAlt | string | `e-INFRA CZ Logo` | Logo alt text |
