# Separator

Separators display a horizontal or vertical divider line to visually separate content sections.

<ComponentPreview>
<div className="flex flex-col gap-6 w-full max-w-sm">
  <div>
    <p className="text-sm font-medium">e-INFRA CZ Design System</p>

    <Separator className="my-4" />
    <div className="flex items-center gap-4 text-sm">
      <span>Blog</span>
      <Separator orientation="vertical" className="h-4" />
      <span>Docs</span>
      <Separator orientation="vertical" className="h-4" />
      <span>Source</span>
    </div>
  </div>
</div>

## Usage

```tsx
import { Separator } from '@e-infra/design-system'

<Separator />
<Separator orientation="vertical" className="h-8" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| orientation | `horizontal` \| `vertical` | `horizontal` | Direction of the separator |
| decorative | boolean | `true` | Whether the separator is decorative |
| className | string | - | Additional CSS classes |
| ...props | React.ComponentProps | - | Radix UI Separator props |

## Orientation

| Orientation | CSS |
|-------------|-----|
| horizontal | `h-px w-full` |
| vertical | `h-full w-px` |
