# Command

Command provides a searchable command palette for accessing app features quickly.

<ComponentPreview>
  <CommandPreview />
</ComponentPreview>

## Usage

```tsx
import { Command, CommandDialog, CommandInput, CommandList, CommandEmpty, CommandGroup, CommandItem, CommandShortcut } from '@e-infra/design-system'

<CommandDialog>
  <CommandInput placeholder="Type a command or search..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Suggestions">
      <CommandItem>Calendar</CommandItem>
      <CommandItem>Search Emoji</CommandItem>
      <CommandItem>Calculator</CommandItem>
    </CommandGroup>
  </CommandList>
</CommandDialog>
```

## Components

| Component | Description |
|-----------|-------------|
| `Command` | Main command container |
| `CommandDialog` | Dialog wrapper |
| `CommandInput` | Search input field |
| `CommandList` | Results list container |
| `CommandEmpty` | Empty state component |
| `CommandGroup` | Group of related items |
| `CommandItem` | Individual command |
| `CommandSeparator` | Divider between groups |
| `CommandShortcut` | Keyboard shortcut display |
| `CommandPort` | Portal wrapper |
