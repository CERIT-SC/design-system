# Carousel

Carousels display a series of content items that can be navigated horizontally or vertically.

<ComponentPreview>
  <CarouselPreview />
</ComponentPreview>

## Usage

```tsx
import { Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext } from '@e-infra/design-system'

<Carousel className="w-full max-w-xs">
  <CarouselContent>
    <CarouselItem>Slide 1</CarouselItem>
    <CarouselItem>Slide 2</CarouselItem>
    <CarouselItem>Slide 3</CarouselItem>
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Components

| Component | Description |
|-----------|-------------|
| `Carousel` | Main container with context |
| `CarouselContent` | Scrollable content wrapper |
| `CarouselItem` | Individual slide |
| `CarouselPrevious` | Previous slide button |
| `CarouselNext` | Next slide button |

## Props

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| opts | object | - | Embla carousel options |
| plugins | array | - | Embla carousel plugins |
| orientation | `horizontal` \| `vertical` | `horizontal` | Slide direction |
| setApi | (api: CarouselApi) => void | - | Set carousel API reference |
| className | string | - | Additional CSS classes |

### CarouselPrevious/CarouselNext

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | Button variant | `outline` | Button style |
| size | Button size | `icon` | Button size |
| className | string | - | Additional CSS classes |
| ...props | Button props | - | Button component props |
